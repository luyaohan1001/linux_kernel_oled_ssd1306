◆ FONT_TABLE

const unsigned char FONT_TABLE[][FONT_TABLE_CHAR_WIDTH]
static

ASCII Font table defined in hex encoding.

Note
This table is accessed through numerical value of a char. Each single char is rendered on screen byte by byte (per slice). Non-Alphanumeric characters are encoded 0.
