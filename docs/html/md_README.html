To compile.

Setup compile environment
$ sudo make setup

Compile
$ sudo make

Successful compile message example:

To run:

Insert the kernel module

    $ sudo make insmod

To remove the kernel module:

    $ sudo make rmmod

To check for printk log:

    $ dmesg

To generate docs by doxygen

    $ make doxygen

    $ cd /docs/html
