A\+S\+C\+II Font table defined in hex encoding. 

\begin{DoxyNote}{Note}
This table is accessed through numerical value of a char. Each single char is rendered on screen byte by byte (per slice). Non-\/\+Alphanumeric characters are encoded 0; they are meaningless for printing but including them avoids remapping when interpreting ascii numeric value as the access index to this table. 
\end{DoxyNote}
